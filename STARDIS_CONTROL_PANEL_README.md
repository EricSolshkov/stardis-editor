# Stardis 控制面板使用说明

## 概述

Stardis 控制面板是一个图形化界面工具，用于配置和运行 Stardis 热辐射计算软件。该面板提供了对 Stardis 所有命令行参数的可视化控制，简化了复杂参数的配置过程。

## 主要功能

### 1. 可执行文件配置

在面板顶部可以配置：
- **Stardis.exe 路径**: 指定 stardis.exe 可执行文件的位置
- **工作目录**: 指定命令行运行的工作目录（相对路径将基于此目录）

### 2. 基本参数（必需）

#### 模型文件 (-M) *必需
- 指定 Stardis 计算所需的模型文件
- 支持多个模型文件（根据原始代码逻辑）

#### Monte Carlo 样本数 (-n)
- 设置 Monte Carlo 模拟的样本数量
- 默认值: 1,000,000
- 范围: 1 到 1,000,000,000

#### 并行计算 (-t)
- 设置并行计算使用的线程数
- 默认值: 4
- 范围: 1 到 256

#### 详细度 (-V)
- 控制日志输出的详细程度
- 级别:
  - 0: 静默模式
  - 1: 基本信息
  - 2: 详细信息
  - 3: 调试信息

### 3. 计算模式

#### 体积温度探针 (-p)
- 在指定的三维坐标位置计算温度
- 参数:
  - 位置 (x, y, z): 探针的三维坐标
  - 时间范围 (可选): t1, t2 指定时间窗口

#### 表面温度探针 (-P)
- 在表面指定位置计算温度
- 参数:
  - 位置 (x, y, z): 探针的三维坐标
  - 侧面标识: 例如 +x, -y, +z 等

#### 表面通量密度探针 (-f)
- 计算表面指定位置的通量密度
- 参数:
  - 位置 (x, y, z): 探针的三维坐标

#### 介质平均温度 (-m)
- 计算指定介质中的平均温度
- 参数:
  - 介质名称: 在模型中定义的介质名称
  - 时间范围 (可选)

#### 表面平均温度 (-s)
- 计算表面的平均温度
- 参数:
  - 求解文件: 指定表面定义文件

#### 表面温度图 (-S)
- 生成表面的温度分布图
- 参数:
  - 求解文件: 指定表面定义文件

#### 表面通量 (-F)
- 计算穿过表面的总通量
- 参数:
  - 求解文件: 指定表面定义文件

#### 红外图像渲染 (-R)
- 渲染红外热图像
- 参数:
  - 相机配置: 相机参数配置字符串

### 4. 高级选项

#### 扩散算法 (-a)
- 选择扩散算法:
  - 无: 不使用扩散算法
  - dsphere: Delta-sphere 算法
  - wos: Walk-on-Spheres 算法

#### Picard 迭代阶数 (-o)
- 设置 Picard 迭代的阶数
- 默认值: 1
- 范围: 1 到 100

#### 初始时间 (-I)
- 设置模拟的初始时间
- 用于瞬态计算

#### 禁用内部辐射 (-i)
- 勾选此选项可禁用内部辐射计算

#### 随机数生成器状态
- **输入状态文件 (-x)**: 从文件加载随机数生成器状态
- **输出状态文件 (-X)**: 保存随机数生成器状态到文件

#### 扩展结果 (-e)
- 启用扩展结果输出
- 提供更详细的计算结果

### 5. 输出选项

#### 导出模型 (-d)
- 将内部模型数据导出到文件
- 用于调试和验证

#### 导出 C Chunks (-c)
- 导出计算块数据
- 参数:
  - 文件前缀: 输出文件的前缀

#### 导出路径 (-D)
- 导出热传输路径
- 参数:
  - 导出类型: all (所有), error (错误), success (成功)
  - 输出文件: 路径数据保存位置

#### Green 函数输出
- **ASCII 格式 (-g)**: 以 ASCII 格式输出 Green 函数
- **二进制格式 (-G)**: 以二进制格式输出 Green 函数
  - 输出文件: Green 函数数据文件
  - 结束路径文件 (可选): 额外的路径信息

## 使用流程

### 1. 基本使用

1. **配置可执行文件**
   - 点击"浏览..."按钮选择 `stardis.exe` 文件
   - 设置工作目录（可选，默认为程序所在目录）

2. **设置必需参数**
   - 在"基本参数"标签页中，点击"浏览..."选择模型文件
   - 根据需要调整样本数、线程数和详细度

3. **选择计算模式**
   - 在"计算模式"标签页中，勾选需要的计算类型
   - 填写相应的参数（位置、文件名等）

4. **配置高级选项**（可选）
   - 在"高级选项"标签页中设置扩散算法、Picard阶数等

5. **设置输出选项**（可选）
   - 在"输出选项"标签页中配置需要的输出格式

6. **预览和运行**
   - 点击"更新预览"按钮查看完整的命令行
   - 点击"运行 Stardis"按钮执行计算
   - 在"运行日志"区域查看实时输出

### 2. 高级使用

#### 命令预览
在运行前，可以点击"更新预览"查看将要执行的完整命令行，便于调试和验证参数。

#### 日志监控
运行过程中，所有标准输出和错误信息都会实时显示在"运行日志"区域。可以随时点击"清除日志"清空历史记录。

#### 停止运行
如果需要中断正在运行的计算，点击"停止"按钮可以终止进程。

| -G | Green 二进制 | Green函数二进制输出 |

## 注意事项

1. **互斥选项**: 某些计算模式是互斥的，一次只能选择一种主要计算模式
2. **文件路径**: 所有文件路径都支持相对路径和绝对路径，相对路径基于设置的工作目录
3. **Green函数限制**: 使用Green函数输出时，Picard阶数必须为1
4. **红外渲染**: 使用红外图像渲染模式时，样本数(-n)选项无效，应使用渲染配置中的SPP参数

## 集成到主窗口

控制面板可以通过以下方式打开：
1. 点击主窗口右侧的"打开 Stardis 控制面板"按钮
2. 使用菜单栏: Stardis -> 打开控制面板
3. 快捷键: Ctrl+P

## 示例配置

### 示例 1: 简单温度探针计算
```
模型文件: model.stl
样本数: 1000000
线程数: 8
计算模式: 体积温度探针
  位置: (0, 0, 0)
```

### 示例 2: 表面温度分布计算
```
模型文件: model.stl
样本数: 5000000
线程数: 16
计算模式: 表面温度图
  求解文件: surface_mesh.msh
详细度: 2 (详细)
```

### 示例 3: 红外图像渲染
```
模型文件: model.stl
线程数: 16
计算模式: 红外图像渲染
  相机配置: camera_config.txt
输出选项: 导出模型
  输出文件: output_model.dat
```

## 故障排除

### 问题: 无法启动 Stardis
- 检查可执行文件路径是否正确
- 确认 stardis.exe 有执行权限
- 查看日志输出中的错误信息

### 问题: 找不到模型文件
- 确认模型文件路径正确
- 如果使用相对路径，确保工作目录设置正确
- 检查文件是否存在且可读

### 问题: 参数冲突
- 查看命令预览，检查参数组合
- 参考命令行参数对照表，确认参数兼容性
- 查看日志中的详细错误信息
