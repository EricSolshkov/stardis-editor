# Scene Editor v2：组件通信协议

> **阅读层次**：接口契约层，实现者在开始编码前必读。
> 本文档回答：SceneTreePanel、SceneViewport、PropertyPanel 三个组件之间如何传递消息？
> 所有联动通过 Qt 信号/槽机制实现，三个组件不直接相互引用。

---

## 1. 信号定义

```python
# ── 场景树发出的信号 ──────────────────────────────────────────
class SceneTreePanel(QTreeWidget):
    # 选中事件
    body_selected        = pyqtSignal(str)          # body name
    zone_selected        = pyqtSignal(str, str)     # body name, zone name
    connection_selected  = pyqtSignal(str)           # connection name
    probe_selected       = pyqtSignal(str)           # probe name
    camera_selected      = pyqtSignal(str)           # camera name
    light_selected       = pyqtSignal(str)           # light name
    ambient_selected     = pyqtSignal()              # 环境光照节点被选中
    global_selected      = pyqtSignal()              # 全局设置节点被选中
    selection_cleared    = pyqtSignal()

    # 右键操作信号（光源）
    request_add_default_light         = pyqtSignal()
    request_add_spherical_source      = pyqtSignal()
    request_add_spherical_source_prog = pyqtSignal()
    request_delete_light              = pyqtSignal(str)  # light name


# ── 3D 视口发出的信号 ─────────────────────────────────────────
class SceneViewport(QWidget):
    body_picked    = pyqtSignal(str)                     # body name（左键单击）
    probe_picked   = pyqtSignal(str)                     # probe name（左键单击）
    nothing_picked = pyqtSignal()                         # 点击空白
    paint_changed  = pyqtSignal(str, dict)               # body name, {zone_name: cell_ids}
    probe_placed   = pyqtSignal(str, float, float, float)  # body name, x, y, z（双击几何体）


# ── 属性面板发出的信号 ────────────────────────────────────────
class PropertyPanel(QWidget):
    property_changed      = pyqtSignal(str, str, object)  # entity name, property key, new value
    boundary_type_changed = pyqtSignal(str, str, str)     # body name, zone name, new type
```

## 2. 联动矩阵

| 触发事件 | 场景树响应 | 3D 视口响应 | 属性面板响应 |
|---------|-----------|------------|------------|
| 树中单击 Body | 高亮该节点 | Silhouette 轮廓线高亮（黄色线宽 3） | 显示体积属性 |
| 树中单击 Zone | 高亮该节点 | Silhouette 轮廓线 + LUT 区域着色（选中原色，其余暗灰） | 显示边界参数 |
| 树中勾选可见性 | 复选框状态更新 | 对应 actor 显示 / 隐藏 | — |
| 3D 中单击几何体 | 自动展开并选中对应节点 | Silhouette 轮廓线高亮 | 显示体积属性 |
| 3D 中单击探针 | 选中对应 Probe 节点 | 探针标记放大 | 显示探针参数 |
| 3D 中双击几何体 | 在探针组下创建新节点 | 在命中点显示探针标记 | 显示新探针参数 |
| 3D 中拖拽探针 | — | 探针跟随移动 | 实时更新坐标值 |
| 3D 中单击空白 | 取消选中 | 所有对象恢复正常渲染 | 清空或显示全局设置 |
| 属性面板修改探针坐标 | — | 探针标记移动到新位置 | — |
| 属性面板修改参数 | — | 若影响外观则刷新 | — |
| 属性面板改边界类型 | 更新节点图标 / 文字 | 更新区域颜色 | 切换参数面板 |
| 树中单击光源 | 高亮该节点 | — | 显示光源参数（按类型动态显隐字段） |
| 树中单击环境光照 | 高亮该节点 | — | 显示环境光照强度编辑器 |
| 属性面板修改光源参数 | 刷新光源显示文字 | VTK 光源同步更新 | — |
| 属性面板修改环境光照强度 | 刷新强度数值显示 | 所有 Actor 的 Ambient 同步更新 | — |
| 右键光源组 → 添加 | 创建新光源节点 | VTK 添加对应光源 | 显示新光源参数 |
| 右键光源 → 删除 | 移除节点，可能触发 ensure_default | VTK 移除对应光源 | 清空属性面板 |

## 3. 事件流向图

```
用户操作
    │
    ├─→ SceneTreePanel
    │       │  body_selected(name)
    │       │  zone_selected(bname, zname)
    │       │  probe_selected(name)
    │       │  ...
    │       ↓
    │   SceneEditor (主窗口，槽函数中转)
    │       ├─→ SceneViewport.highlight_body(name)
    │       ├─→ SceneViewport.highlight_zone(bname, zname)
    │       └─→ PropertyPanel.show_body(body)
    │
    ├─→ SceneViewport
    │       │  body_picked(name)
    │       │  probe_placed(bname, x, y, z)
    │       │  paint_changed(bname, zone_dict)
    │       ↓
    │   SceneEditor
    │       ├─→ SceneTreePanel.select_node(name)
    │       ├─→ SceneModel.update(...)
    │       └─→ PropertyPanel.show_body(body)
    │
    └─→ PropertyPanel
            │  property_changed(name, key, value)
            │  boundary_type_changed(bname, zname, type)
            ↓
        SceneEditor
            ├─→ SceneModel.update(...)
            ├─→ SceneTreePanel.refresh_node(name)
            └─→ SceneViewport.refresh_actor(name)
```

## 4. SceneEditor 主窗口的职责

SceneEditor 是唯一持有 `SceneModel` 引用的组件，负责：
1. 连接三个面板的信号到自身槽函数
2. 在槽函数中修改 `SceneModel`
3. 将变更结果分发给需要刷新的面板
4. 管理撤销/重做栈（涂选以外的操作）
