# 物理材质管理系统

> **归档层次**：系统架构，面向全局理解与跨系统集成。
> 详细实现参见 [`design/material_database/`](../material_database/)。

---

## 1. 系统定位

物理材质管理系统为场景中的几何体（Body）提供**可复用的命名物理参数集**（导热系数 λ、密度 ρ、比热容 cp），使用户无需为每个几何体手动输入重复的热物性参数。

该系统是**编辑器层面的辅助概念**：材质名称对 Stardis 求解器透明，保存/导出时仅将数值内联到场景文件，不写入材质名称。

---

## 2. 组件结构

```
物理材质管理系统
│
├── 数据层
│   ├── Material             (dataclass)  — 命名材质：name / λ / ρ / cp / category / is_builtin
│   ├── MaterialRef          (dataclass)  — Body 内联材质：λ / ρ / cp / source_material
│   └── MaterialDatabase     (QObject)    — 材质库：CRUD、持久化、内置材质注册
│
├── UI 层
│   ├── BodyEditor 材质选择器            — 下拉框 + "管理…" 按钮（内嵌于 PropertyPanel）
│   └── MaterialManagerDialog (QDialog)  — 完整材质库 CRUD 管理对话框
│
└── 持久化
    └── material_database.json           — 仅存储用户自定义材质（内置材质硬编码）
```

---

## 3. 关键数据流

```
MaterialDatabase (内置25种 + 用户自定义)
        │
        │ 用户选择材质名
        ▼
MaterialRef (λ/ρ/cp 值拷贝到 Body)
        │
        │ 用户手动修改任意参数 → source_material 清空（断开关联）
        │ 用户重新选择材质名   → 重新赋值并设置 source_material
        ▼
Body.volume.material  ←→  PropertyPanel.BodyEditor（UI 双向绑定）
        │
        ▼
SceneWriter → scene.txt（λ/ρ/cp 数值内联写出，不含材质名）
```

---

## 4. 核心设计原则

| 原则 | 说明 |
|------|------|
| **值拷贝语义** | 选择材质时将参数复制到 Body，不持有材质引用；材质库修改不影响已赋值的 Body |
| **关联追踪** | `source_material` 字段记录最后一次关联的材质名，手动改参数后自动置空 |
| **内置不可删** | 内置 25 种材质 `is_builtin=True`，UI 中只读；用户可"复制另存"为自定义材质 |
| **格式透明** | 场景文件（`.txt`）中不写入材质名；向后完全兼容已有工程 |
| **最近使用** | `EditorPreferences.recent_materials` 存储最近 5 种，下拉框顶部快速访问 |

## 5. 内置材质分类（25 种）

| 分类 | 示例材质 |
|------|---------|
| 金属 | Copper, Aluminum, Steel_Stainless, Steel_Mild, Iron, Gold, Brass, Titanium |
| 绝缘体 | Ceramic_Alumina, Concrete, Glass_Borosilicate, PTFE, Epoxy_Resin |
| 流体 | Air_300K, Water_300K, Nitrogen_300K |
| 其他 | Silicon, Graphite, Carbon_Fiber, PCB_FR4, … |

---

## 6. 持久化格式

`material_database.json` 仅保存用户自定义材质（`is_builtin=False`）：

```json
{
  "version": 1,
  "custom_materials": [
    {
      "name": "MyAlloy",
      "conductivity": 45.0,
      "density": 7200,
      "specific_heat": 460,
      "category": "自定义",
      "description": "项目定制合金",
      "is_builtin": false
    }
  ]
}
```

## 7. 与其他系统的接口

| 调用方 | 接口 | 说明 |
|--------|------|------|
| `PropertyPanel.BodyEditor` | `MaterialDatabase.list_all()` / `get(name)` | 填充下拉框、选中时赋值 |
| `SceneTreePanel` | `Body.right_click → save_material / apply_material` | 右键快捷从 Body 保存/应用材质 |
| `SceneWriter` | `Body.volume.material.conductivity/density/specific_heat` | 读取数值写入 scene.txt |
| `EditorPreferences` | `recent_materials: List[str]` | 下拉框顶部最近使用列表 |
