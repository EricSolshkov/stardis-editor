# 几何变换系统设计文档

本目录包含几何变换（Geometry Transform）子系统的设计文档。该系统允许用户对 STL 几何体施加平移 / 旋转 / 缩放变换，并在保存时将变换后的网格导出为 Stardis 所需的 STL 格式。

---

|------|-----------|------|
| [01_设计方案.md](01_设计方案.md) | 完整架构、数据模型、UI、IO 管线、实现计划 | 所有人 |

---

## 核心概念

- **raw_model/**：场景目录下的子目录，存放未经变换的原始 STL 文件
- **GeometryTransform**：TRS 变换数据类（平移 → 旋转 → 缩放），四元数内部表示，UI 欧拉角
- 从 `.txt` 直接加载的场景：变换锁定为单位变换，不可编辑
