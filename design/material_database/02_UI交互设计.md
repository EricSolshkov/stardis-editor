```

│ 类型:        [SOLID ▼]                           │
│                                                  │
│ 材质:        [Copper           ▼] [管理...]       │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │
│ 导热系数 λ:   [400.0000    ] W/m/K               │
│ 密度 ρ:      [8960.00     ] kg/m³                │
│ 比热容 cp:    [385.00      ] J/kg/K               │
│                                                  │
│ 随机行走步长 δ: [AUTO ▼] [0.001000] m             │
│ 初始温度:     [300.0       ] K                    │
│ 施加温度:     [UNKNOWN ▼]  [300.0   ] K           │
│ 体积热源:     [0.00        ] W/m³                 │
│ 法线朝向:     [FRONT ▼]                           │
└──────────────────────────────────────────────────┘
```

### 1.2 材质下拉框 (QComboBox)

**内容组织**：
```
── 最近使用 ──
  Copper
  Steel_Stainless
── 金属 ──
  Aluminum
  Brass
  Copper
  Gold
  Iron
  ...
── 绝缘体 ──
  Ceramic_Alumina
  Concrete
  ...
── 流体 ──
  Air_300K
  Water_300K
  ...
── 自定义 ──
  MyAlloy
  ...
── ──────── ──
  (自定义值)          ← 手动输入模式，不关联任何材质
```

**行为规则**：
- 选中命名材质 → 自动填充 λ/ρ/cp spinbox，`source_material` 记录名称
- 选中"(自定义值)" → spinbox 保持当前值不变，`source_material` 清空
- 手动修改任意 spinbox → 下拉框自动切换到"(自定义值)"，`source_material` 清空
- 加载 Body 时：若 `source_material` 非空且在库中，下拉框选中对应项；否则显示"(自定义值)"

**最近使用**：
- 记录最近 5 个被选用的材质名称
- 存储在 `EditorPreferences` 中（`recent_materials: List[str]`）
- 便于快速重复应用同一材质

### 1.3 "管理..." 按钮

点击打开 `MaterialManagerDialog`（见第 2 节）。

## 2. MaterialManagerDialog

### 2.1 整体布局

```
┌── 材质库管理 ─────────────────────────────────────────────────┐
│                                                               │
│ ┌─ 分类 ─────┐ ┌─ 材质列表 ──────────────────────────────┐    │
│ │ 全部       │ │ 名称          │ λ W/m/K │ ρ kg/m³ │ cp  │    │
│ │ 金属       │ │ ● Copper      │  400    │ 8960   │ 385 │    │
│ │ 绝缘体     │ │ ● Aluminum    │  237    │ 2700   │ 900 │    │
│ │ 流体       │ │ ○ Steel_Mild  │   50    │ 7850   │ 500 │    │
│ │ 其他       │ │ ○ MyAlloy     │   45    │ 7200   │ 460 │    │
│ │ 自定义      │ │               │         │        │     │    │
│ └────────────┘ └──────────────────────────────────────────┘    │
│                                                               │
│ ┌─ 材质详情 ────────────────────────────────────────────────┐  │
│ │ 名称:      [Copper           ]                            │  │
│ │ 分类:      [金属        ▼]                                │  │
│ │ 导热系数 λ: [400.0000        ] W/m/K                      │  │
│ │ 密度 ρ:    [8960.00          ] kg/m³                      │  │
│ │ 比热容 cp:  [385.00           ] J/kg/K                     │  │
│ │ 备注:      [纯铜                              ]           │  │
│ │                              [保存修改]                    │  │
│ └────────────────────────────────────────────────────────────┘  │
│                                                               │
│ [新建材质]  [复制]  [删除]  [从文件导入...]  [导出到文件...]    │
│                                                    [关闭]     │
└───────────────────────────────────────────────────────────────┘
```

### 2.2 组件说明

| 区域 | 组件 | 说明 |
|------|------|------|
| 分类列表 | `QListWidget` | 左侧分类过滤器，选中分类后右侧列表过滤。"全部"显示所有材质 |
| 材质列表 | `QTableWidget` | 中部表格，显示 name / λ / ρ / cp，可排序。● = 内置，○ = 用户自定义 |
| 材质详情 | `QFormLayout` | 下部编辑区，选中材质时填充。内置材质的名称/分类/数值字段禁用（只读） |
| 底部按钮 | `QPushButton` × 5 | 新建/复制/删除/导入/导出 |

### 2.3 操作行为

| 操作 | 触发 | 行为 |
|------|------|------|
| **新建** | 点击"新建材质" | 在详情区清空字段，名称设为 "New_Material"，聚焦名称输入框。用户编辑后点"保存" |
| **复制** | 选中材质 → 点击"复制" | 以 `{name}_copy` 为新名称复制材质，自动标记为自定义 |
| **编辑** | 选中材质 → 修改详情区 | 内置材质：数值字段禁用，仅备注可编辑。用户自定义：全部可编辑 |
| **保存修改** | 点击"保存修改" | 验证名称唯一性 → 更新数据库 → 刷新列表 |
| **删除** | 选中用户材质 → 点击"删除" | 确认对话框 → 删除。内置材质禁用删除按钮 |
| **导入** | 点击"从文件导入..." | `QFileDialog` 选择 `.json` 文件，解析后合并到自定义材质，名称冲突提示覆盖/跳过/重命名 |
| **导出** | 点击"导出到文件..." | 导出选中材质或全部自定义材质为 `.json` 文件 |

### 2.4 验证规则

- 名称：非空、唯一、仅允许字母数字下划线（同 Body name 规则）
- λ ≥ 0（流体可为 0）
- ρ > 0
- cp > 0

## 3. 场景树右键菜单扩展

### 3.1 Body 节点右键

在现有右键菜单中新增：

```
Body: FOAM
├── ...（现有项）
├── ──────────
├── 应用材质          ▶  ┌─ Copper ─────────┐
│                       │ Aluminum          │
│                       │ Steel_Stainless   │
│                       │ ...               │
│                       │ ────────────────  │
│                       │ 浏览材质库...      │
│                       └───────────────────┘
├── 保存当前材质到库...
└── ...
```

| 菜单项 | 行为 |
|--------|------|
| **应用材质 →** | 子菜单列出最近使用 + 全部材质。选中后应用到当前 Body |
| **保存当前材质到库...** | 弹出简易对话框（名称+分类），将当前 Body 的 λ/ρ/cp 保存为新材质 |

### 3.2 多选 Body 批量应用

场景树支持多选 Body 时：
- 右键 → "应用材质" 同时修改所有选中 Body 的材质
- PropertyPanel 显示多选状态时，材质下拉框显示"(多个值)"或公共材质名

## 5. 交互流程图

### 5.1 新建 Body 时应用材质

```
用户: Scene → Add Body → 选择 STL
  │
  ├── Body 创建（默认 MaterialRef(1,1,1)）
  │
  ├── SceneTree 选中新 Body → PropertyPanel 切换到 BodyEditor
  │
  ├── 用户点击材质下拉 → 选择 "Copper"
  │
  ├── λ=400, ρ=8960, cp=385 自动填入 spinbox
  │
  ├── source_material = "Copper"
  │
  └── property_changed → SceneEditor dirty = True
```

### 5.2 从现有场景加载

```
用户: File → Open → scene.txt
  │
  ├── SceneParser 解析 SOLID → Body(material=MaterialRef(50, 7850, 500))
  │
  ├── load_project → 读取 body_materials: {"SHELL": "Steel_Mild"}
  │
  ├── Body.material.source_material = "Steel_Mild"
  │
  ├── BodyEditor.load(body)
  │   ├── spinbox 显示 50 / 7850 / 500
  │   └── 下拉框选中 "Steel_Mild"（若值匹配），否则显示 "(自定义值)"
  │
  └── 用户可继续编辑或切换到其他材质
```

> **值匹配检查**：加载时除了看 `source_material` 名称，还需验证当前 λ/ρ/cp
> 是否与数据库中该材质的值一致。若不一致（用户可能手动改过），显示
> "(自定义值 / 原 Steel_Mild)"以提示偏差。

### 5.3 保存流程

```
用户: File → Save
  │
  ├── SceneWriter.write_scene() → scene.txt（λ/ρ/cp 内联数值，无变化）
  │
  ├── SceneModel.save_project() → .stardis_project.json
  │   └── 新增 body_materials 段
  │
  └── MaterialDatabase.save() → material_database.json（若有自定义材质变更）
```

## 6. 视觉风格

### 6.1 材质下拉框样式

- 内置材质名前缀 `●`（实心圆），用户材质前缀 `○`（空心圆）
- 分类分隔线用 `QComboBox` 的 separator 项
- 当前项为"(自定义值)"时，文字显示为灰色斜体

### 6.2 内置材质只读指示

- MaterialManagerDialog 中，选中内置材质时：
  - 名称、分类、λ/ρ/cp 输入框变为灰色禁用状态
  - 删除按钮禁用
  - 底部提示文字："内置材质不可修改。可使用"复制"创建自定义副本。"
