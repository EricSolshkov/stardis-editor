## 1. 场景树面板

### 1.1 树形结构示例

```
场景树 (Porous 场景示例)
├── 🌐 全局设置
│   ├── TRAD: 680 / 800
│   └── SCALE: 1.0
│
├── 🧊 几何体
│   └── FOAM [SOLID]
│       ├── 🔴 LAT [对流] (780 △)
│       ├── 🔵 T0 [固定温度 750K] (212 △)
│       └── 🔵 T1 [固定温度 850K] (212 △)
│
├── 🔗 连接 (0)
│
├── 📍 探针 (3)
│   ├── 📍 P1 [体积温度] (0.5, 0.5, 0.5) t=INF
│   ├── 📍 P2 [表面温度] (0.1, 0.0, 0.05) side=FRONT
│   └── 📍 F1 [表面通量] (0.0, 0.05, 0.05)
│
├── 📷 摄像机 (IR 模式)
│   └── Camera1: pos → tgt
│
└── 💡 光源 (1)
    ├── ☀ 环境光照 (强度 0.15)
    └── DefaultLight [默认]

场景树 (City 场景示例)
├── 🌐 全局设置
│
├── 🧊 几何体
│   ├── WALL [SOLID]
│   │   ├── 🔴 EXTERNAL [对流 T=273K]
│   │   └── 🔴 INTERNAL [对流 T=293K]
│   ├── GLAZING [SOLID]
│   │   ├── 🟠 EXT_GLAZING [对流 ε=0.15 spec=0.8]
│   │   └── 🟠 INT_GLAZING [对流 ε=0.15 spec=0.8]
│   └── GROUND [SOLID]
│
└── 💡 光源 (2)
    ├── ☀ 环境光照 (强度 0.15)
    ├── SphericalSource1 [常量球面源]
    └── SphericalSourceProg1 [可编程球面源]
```

**要点**：
- 表面区域节点直接挂在 Body 下，无表面区域组节点
- 表面区域节点由涂选编辑自动生成，不手动创建/删除
- 覆盖率等统计信息显示在属性面板中，不作为树节点

### 1.2 节点类型与操作

| 节点类型 | 图标 | 单击 | 双击 | 右键菜单 |
|---------|------|------|------|---------|
| 全局设置 | 🌐 | 属性面板显示全局参数 | — | — |
| 几何体组 | 📁 | — | — | 添加几何体 |
| 几何体 (Body) | 🧊 | 3D 高亮整个几何体 + 属性面板显示体积属性 | 重命名 | 进入涂选编辑、删除 |
| 表面区域 | 🔴🔵🟢🟣 | 3D 高亮该区域 + 属性面板显示边界参数 | 重命名 | — (不可删除/不可修改范围) |
| 连接 | 🔗 | 高亮界面几何 + 显示连接参数 | 重命名 | 删除 |
| 探针组 | 📍 | — | — | 添加探针 |
| 探针 | 📍 | 属性面板显示参数 + 3D 视口高亮 | 重命名 | 删除 |
| 摄像机 | 📷 | 显示摄像机锥体 + 属性面板显示参数 | 重命名 | 删除、使用当前视角 |
| 光源组 | 💡 | — | — | 添加默认光源、添加常量球面源、添加可编程球面源 |
| 环境光照 | ☀ | 属性面板显示环境光照强度 | — | — (不可删除，恒存在) |
| 光源 (SceneLight) | — | 属性面板显示光源参数 | — | 删除 |

**关键约束**：
- 几何体的 STL 设置、材质赋值等均在**属性面板**中完成，不在右键菜单
- 表面区域节点由涂选模式退出时**自动生成**（根据 BoundaryLabel 数组中的不同值），不支持手动添加或删除
- 表面区域的范围（包含哪些三角面）只能通过重新进入涂选模式修改，不能在树中直接操作
- 几何体不支持复制

### 1.3 拖拽操作

| 操作 | 效果 |
|------|------|
| 外部 STL 文件拖入「几何体」组 | 创建新几何体 |

不支持：拖入 STL 到几何体创建表面区域（表面区域只能通过涂选模式生成）、拖拽排序、拖动表面区域。

### 1.4 覆盖率信息

覆盖率统计显示在属性面板中（选中 Body 时的「表面区域概览」部分），不作为场景树节点。

```
覆盖率计算 = Σ(所有 surface_zone 的三角面数) / 母体总三角面数
```

| 状态 | 显示 | 含义 |
|------|------|------|
| 100% | ✅ | 所有表面已分配边界条件 |
| 0% < x < 100% | ⚠️ 78% | 部分表面未分配（灰色区域可见） |
| 0% | ❌ | 无任何边界条件 |
| > 100% | ⛔ | 区域有重叠 |

> **注意**：覆盖率不足 100% 不一定是错误。某些面可能不需要边界条件（由 stardis 的域划分自动处理为内部界面），但编辑器应提醒用户检查。

## 2. 3D 视口交互

### 2.1 视口模式

视口有两种互斥的工作模式，通过工具栏按钮切换：

| 模式 | 鼠标操作 | 用途 |
|------|---------|------|
| **导航模式** (默认) | 中键按住旋转、WASD+Shift/Ctrl 平移导航、滚轮缩放、左键点选、双击放置探针 | 浏览场景、选择对象、放置探针 |
| **涂选模式** | 左键拖拽涂选、Ctrl+左键擦除、中键按住旋转、滚轮缩放 | 在几何体表面划分边界区域 |

### 2.2 导航模式：选择 (Picking)

| 操作 | 行为 |
|------|------|
| 左键单击几何体 | 选中该 Body → 场景树同步高亮 + 属性面板切换 |
| 左键单击探针标记 | 选中该 Probe → 场景树同步 + 属性面板切换 |
| 左键单击空白 | 取消选择 |
| 双击几何体表面 | 在命中点放置新探针 |

不支持多选。左键点选仅覆盖 Body 和 Probe，不直接选中表面区域（表面区域通过场景树查看）。

### 2.3 导航模式：高亮反馈

高亮效果**仅在导航模式下显示**，进入涂选模式时自动清除。

| 选中对象 | 3D 视觉效果 |
|---------|------------|
| 几何体 (Body) | **轮廓线高亮 (Silhouette)**：黄色轮廓线，线宽 3，自动跟随相机视角更新 |
| 表面区域 (SurfaceZone) | **轮廓线 + 区域着色高亮**：所属 Body 显示轮廓线；选中区域以其分配的颜色实体着色，其余区域变为暗灰色 (0.45, 0.45, 0.45)，取消选择时恢复原色 |
| 探针 | 标记图标放大 + 坐标标注 |
| 无选中 | 所有对象恢复正常渲染 |

> 实现细节见 [04_VTK实现方案.md → §1 选择高亮](04_VTK实现方案.md)

### 2.4 导航模式：法线显示

选中几何体后，可切换其面法线可视化，用于检查法线朝向（影响材质 `side` 设置）。

| 操作 | 行为 |
|------|------|
| `N` 键 | 切换选中几何体的法线显示（开/关） |
| `+` 键 | 法线显示长度 × 10 |
| `-` 键 | 法线显示长度 × 0.1 |

**视觉效果**：每个三角面中心向法线方向延伸一条红色线段（默认长度 1）。长度缩放对所有已显示法线的几何体同时生效。切换场景或清除选择时自动移除所有法线显示。

> 实现细节见 [04_VTK实现方案.md → §2 法线显示](04_VTK实现方案.md)

### 2.5 导航模式：悬停提示 (Tooltip)

鼠标悬停在 3D 对象（几何体或探针）上时，显示对象名称。

### 2.6 涂选模式：进入与退出

**前提条件**：必须先选中一个几何体 (Body)。

**进入方式**：
- 快捷键 `B` (Brush)
- 右键菜单「涂选编辑」

**进入后**：
- 场景中其他几何体自动变为半透明，仅当前几何体可操作
- 几何体表面按已有 SurfaceZone 着色，未分配区域显示为灰色
- 视口顶部出现工具条

**退出方式**：
- 右键单击 → 退出涂选模式，确认修改，更新 SceneModel 中的 surface_zones
- ESC 键 → 取消涂选，丢弃所有修改，恢复进入前状态

> **持久化说明**：退出涂选模式（右键确认）后，涂选数据保存在内存中的 SceneModel。只有用户执行「保存」操作时，涂选结果才被导出为 B_*.stl 并持久化到磁盘。如果关闭程序前未保存，涂选状态丢失是符合预期的行为。

### 2.7 涂选模式：工具条

```
┌──────────────────────────────────────────────────────────────────────────────────────┐
│ 当前区域: [LAT - 对流 ▼] [+ 新建区域] │ [🖌 画刷] [▣ 全部] [🌊 洪泛] [◈ 替换] │ [↩] [↪] │
└──────────────────────────────────────────────────────────────────────────────────────┘
```

- **当前区域下拉**: 选择要涂选的目标 SurfaceZone
- **新建区域**: 创建一个新 SurfaceZone（自动分配 zone_id）并设为当前区域
- **画刷 / 全部 / 洪泛 / 替换**：四种互斥画刷模式（见 §2.8）
- **撤销/重做**: 回退涂选操作（等同 Ctrl+Z / Ctrl+Y）

### 2.8 涂选模式：画刷模式（四种互斥）

工具栏中的四个画刷模式按钮互斥，默认选中**画刷**模式。

Ctrl 修饰符对所有模式通用：将目标区域设为"未分配"（zone_id = 0）而非当前区域。

#### 画刷模式（Brush）

- 左键按住拖拽 → 画刷覆盖范围内的三角面标记为当前区域
- Ctrl + 左键拖拽 → 擦除（三角面恢复为"未分配"）
- 滚轮 → 调整画刷大小

**画刷大小**：以屏幕空间像素为单位的圆形区域，最小为单个三角面（点选）。画刷大小指屏幕上的像素半径，不是世界坐标距离——这样在任何缩放级别下操作感一致。

#### 全部填充模式（Fill All）

- 单击任意三角面 → 将整个几何体的**所有**三角面设为当前区域
- Ctrl + 单击 → 将所有三角面设为"未分配"

适用场景：集中局部分区域已涂好，需要将剩余解领一次性全部转移为一个区域时使用。

#### 洪泛填充模式（Flood Fill）

- 单击任意三角面 → 以该三角面为起点，沿三角面邻接关系进行 BFS 漫延
  — 仅将与起点 **zone_id 相同**且**尚都连通**的三角面，设为当前区域
- Ctrl + 单击 → 洪泛擦除（连通浮棋区域→未分配）

适用场景：已途尚未局部涂选的连通区域（如多个尚未分配的几何连送体）。

> BFS 仅在共边的三角面间传播（整数"共有2个顶点" = 共边）。打就列邻不过共顶点不共边的邻接三角形。

#### 替换填充模式（Replace Fill）

- 单击任意三角面 → 将整个几何体内所有 zone_id 相同（无论是否连通）的三角面，设为当前区域
- Ctrl + 单击 → 替换擦除（同 zone_id 的所有三角面→未分配）

适用场景：一个生理区域由多次涂抹形成了多尚不连岛屿，想一次性将它们全部归并到另一个区域。

> 实现细节见 [04_VTK实现方案.md → §7 涂选画刷与填充](04_VTK实现方案.md)

### 2.9 涂选模式：区域颜色系统

每个 SurfaceZone 绑定一个与其 zone_id 对应的显示颜色。**相同 BC 类型但参数不同的区域拥有独立的 zone_id 和颜色**。

**默认配色规则**（新建区域时自动分配）：

```
未分配 (zone_id=0) → 浅灰色 (0.7, 0.7, 0.7)，半透明 α=0.5
T_BOUNDARY      → 蓝色系 (H≈0.60)    "恒温 — 冷色调"
H_BOUNDARY      → 红/橙系 (H≈0.05)    "换热 — 暖色调"
F_BOUNDARY      → 绿色系 (H≈0.33)    "能量输入"
HF_BOUNDARY     → 紫色系 (H≈0.80)    "组合"
同类型第 N 个区域  → 在基色调内调整明度/饱和度，与同类型其他区域区分
```

**用户自定义颜色**：属性面板（选中区域 → 显示颜色方块 → 点击修改）可为任意区域覆盖配色。修改后立即刷新 LUT。

**画刷实时预览**：当前画刷覆盖的候选单元显示前香色 (alpha 0.5，小幅闪烁) 与当前区域目标颜色叠加，不破坏已有着色。

### 2.10 涂选模式：撤销/重做

- 每次鼠标释放 (mouseUp) 生成一个撤销点
- Ctrl+Z / Ctrl+Y 或工具栏按钮触发
- 撤销栈深度：50 步

### 2.11 探针放置与交互

探针在**导航模式**下操作，不需要进入涂选模式。

**放置**：在导航模式下，**双击几何体表面**即可在该位置放置一个探针：
- 自动递增名称（P1, P2, P3...）
- 默认类型为体积温度探针（`-p`），可在属性面板中切换
- 体积温度探针：放置在命中点（表面上），编辑器提示「注意：体积探针应在几何体内部，当前在表面上」
- 表面温度/通量探针：放置在命中点，自动吸附到最近的三角面中心

**拖拽移动**：选中探针后：
- 左键按住探针标记 → 拖拽
- 拖拽时探针沿鼠标投影方向移动（投影到与相机平面平行的平面上）
- Shift + 拖拽 → 仅沿 X/Y/Z 轴移动（最近轴）
- 松开鼠标 → 更新坐标，属性面板实时同步

**探针标记样式**：

```
📌 体积温度探针 → 球体 (黄色)，Billboard 文字标签
📌 表面温度探针 → 圆锥底部贴表面 (橙色)，指示法线方向
📌 表面通量探针 → 箭头 (青色)，指向法线方向

自适应尺寸: 标记半径 = 场景包围盒对角线 × 0.01
选中态: 标记放大 + 坐标标注
```

**探针与几何体的关系提示**（属性面板中显示）：

| 提示 | 含义 |
|------|------|
| 在 FOAM 内部 (距表面 0.023m) | 体积探针位置合理 |
| 在 FOAM 表面上 | 表面探针位置合理 |
| 不在任何几何体内 | 体积探针可能无效，警告 |
| 在两个几何体的重叠区 | 提示可能存在歧义 |

> 检测实现见 [04_VTK实现方案.md → §5 探针位置检测](04_VTK实现方案.md)

### 3.3 切换边界类型时的参数变化

| 切换到类型 | 显示的参数 |
|-----------|-----------|
| 固定温度 (T_BOUNDARY) | 仅：温度 T |
| 对流+辐射 (H_BOUNDARY) | Tref、ε、镜面分数、hc、T_env |
| 热通量 (F_BOUNDARY) | 仅：通量 flux (W/m²) |
| 组合 (HF_BOUNDARY) | H_BOUNDARY 全部参数 + flux |

## 4. 从零新建场景的完整工作流

```
1. 新建空场景
   → 全局设置默认值: TRAD 300 300, SCALE 1.0
   → 场景树仅显示「全局设置」节点

2. 导入几何体
   → 文件对话框 / 拖拽 STL 进入视口
   → 创建 Body 节点，type 默认 SOLID
   → 几何体在 3D 视口中显示为灰色（无边界条件）

3. 设置体积属性
   → 选中几何体 → 属性面板
   → 选择材质（从库或手动输入 λ/ρ/cp）
   → 设置温度、法线朝向等

4. 划分表面区域（两种方式）

   方式 A: 导入已有边界 STL
   → 右键几何体 →「导入边界 STL」→ 选择 B_*.stl
   → 自动创建 SurfaceZone，着色显示
   → 属性面板设置边界类型和参数

   方式 B: 涂选模式
   → 选中几何体 → 按 B 进入涂选模式
   → 新建区域 "T0"，选择类型「固定温度」→ 用画刷涂选底面
   → 新建区域 "T1"，选择类型「固定温度」→ 画刷涂选顶面
   → 新建区域 "LAT"，选择类型「对流」→ 画刷涂选剩余所有面
   → 检查覆盖率 100%
   → 右键退出涂选模式（确认）

5. 添加探针 (可选)
   → 在 3D 视口中双击几何体表面 → 自动放置探针
   → 选择探针类型（体积温度/表面温度/表面通量）
   → 设置参数（时刻/面标识）
   → 可拖拽调整位置，属性面板实时更新坐标

6. 添加连接 (可选)
   → 右键「连接」节点 →「新建连接」
   → 选择两个几何体 + 导入界面 STL

7. 验证 → 保存
   → Body → SOLID/FLUID 行
   → SurfaceZone(ImportedSTL) → BOUNDARY 行 (引用原 STL)
   → SurfaceZone(PaintedRegion) → 导出新 STL → BOUNDARY 行
   → Connection → CONNECTION 行
   → Probe → 不写入 scene.txt，保存到 .stardis_project.json
```

> **注意**：探针不属于 stardis 场景文件 (`.txt`)，而是求解器命令行参数 (`-p`/`-P`/`-f`)。
> 探针数据持久化在编辑器项目文件中，提交求解任务时由任务面板自动读取。
