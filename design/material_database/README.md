## 设计目标

- 提供可复用的命名材质，避免用户重复输入 λ/ρ/cp
- 内置常用工程材质（金属、流体、绝缘体等）
- 支持用户自定义材质与分类管理
- 与现有场景编辑器无缝集成，保持向后兼容
