## 子系统关系

```
                    ┌──────────────────────┐
                    │   Scene Editor v2    │
                    │  （主窗口协调者）    │
                    └──────────────────────┘
                         │         │
          ┌──────────────┤         ├──────────────────┐
          ▼              ▼         ▼                   ▼
  ┌─────────────┐  ┌──────────┐ ┌──────────┐  ┌───────────────┐
  │ 物理材质    │  │ 场景数   │ │ 界面编   │  │ 任务链调度    │
  │ 管理系统    │  │ 据模型   │ │ 辑系统   │  │ 系统          │
  │（辅助）     │  │（核心）  │ │（核心）  │  │（执行）       │
  └─────────────┘  └──────────┘ └──────────┘  └───────────────┘
         │               │                            │
         │ 值拷贝赋值     │ SceneModel                 │ 引用名 → CLI
         └──────────────►│◄───────────────────────────┘
                         │
                    scene.txt / STL
                    .stardis_project.json
```

**依赖方向**：
- 材质管理系统 → `Body.volume.material`（单向，值拷贝）
- 任务调度系统 → `SceneModel.cameras/probes`（引用求值，只读）
- 场景编辑系统 → 包含其余两个系统的入口点
